# PLAN - Project Roadmap

## שלבי עבודה
1. **הכנת סביבת עבודה:** הגדרת Google Cloud ו-GitHub (בוצע).
2. **פיתוח בסיסי:** כתיבת קוד ראשוני לחיבור ל-API של Gmail ו-Calendar.
3. **מימוש לוגיקה:** הגדרת הפעולות האוטומטיות של הסוכן.
4. **בדיקות:** הרצת הסוכן ובדיקה שהוא מבצע את הפעולות ביומן ובאימייל.
5. **סיכום:** כתיבת תיעוד סופי.
