# My Gmail Assistant Project

## About
This project is an automated Gmail assistant designed to streamline productivity. It integrates with Google APIs to manage emails, analyze their content, and synchronize relevant tasks with the Google Calendar.

## Project Structure
*   **`my-gmail-bot/`**: Contains the core Python scripts for email processing and API authentication.
*   **`PLAN/`**: Contains project planning documents and architectural roadmaps.
*   **`PRD/`**: Contains Product Requirement Documents detailing the bot's features and logic.
*   **`TODO/`**: Tracks upcoming tasks, feature requests, and bug fixes.

## Features
*   **Email Analysis**: Automatically reads and parses incoming emails.
*   **Calendar Integration**: Creates calendar events based on parsed email data.
*   **Automation**: Efficient management of communication workflows using PowerShell and Python scripts.

## Setup Instructions
1.  Clone the repository.
2.  Ensure you have your `credentials.json` file in the `my-gmail-bot/` directory (ensure this file is **never** pushed to GitHub).
3.  Install the required dependencies.
4.  Run `main.py` to start the assistant.
